Turns out, I was missing a quotation mark in my css file (which you can check). So, yay, we got fonts! We'll pick this arbitrary "Raleway" font for now. The font change really pulls the site together--switching from Times New Roman to a nicer, more "modern"/"minimalist" font seems like a priority if you want your site to stop looking pathetic.

We've got two main issues with the layout that I can't solve: the first is apparent: the paragraph elements and content div extend all the way to the left. I don't totally hate this--it'll have to go eventually, but for now it doesn't look THAT awful. Kind of like I'm talking to someone in a messenger client. The second issue is, if the page is made narrow, the content spits out at the bottom of the page, below the ostensibly 100% height sidebar.

Oh, CSS. I have a feeling we're going to be enemies.
