Design

Of course, the color design from yesterday needs to go. I've been looking at ways to quickly swap out colors, and using something like Paint or Adobe Illustrator to mock up the site would almost surely be the easiest way to do that. But, even before I try that, I don't know exactly what I'm doing--what color should my header/sidebar be?

Fifteen minutes later, I've settled on the colors you see now. This looks pleasant enough. I've defined a variable for each component of the page, so trying out new colors is pretty simple. I will probably continue to tweak this occasionally.

Next on the list...oh okay, a logo! Time to get crackin' in Illustrator. Threw together something geometric and slapped it onto my template page. Adding it to the header requires some ugly CSS work, but I think I can make it happen.

I noticed that on Firefox, the Raleway font doesn't show up. Drat. Not enough time left in the day to fix that today. But, otherwise, I'm pretty happy with this! I'm anticipating tomorrow will be fun.
